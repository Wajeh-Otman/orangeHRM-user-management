*** Please note that the project is still under development ***

This project tests the User Management functionality for the Admin role in the OrangeHRM (https://opensource-demo.orangehrmlive.com/web/index.php/auth/login).
The project includes:
  - Functional Requirements Document (FRD) file
  - Software Test Plan (STP) file
  - Software Test Design (STD) file
  - Software Test Report (STR) file
  - Python-based QA automation project using PLAYWRIGHT
